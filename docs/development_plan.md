# SceneWeaver 开发计划

## 项目概述

SceneWeaver是一个基于Python的端侧实时混合现实创作系统，专为高通骁龙X Elite平台优化设计。本计划详细规划了项目的开发阶段、里程碑和未来发展方向。

## 技术栈确认

### 当前技术栈
- **核心语言**: Python 3.12
- **图形渲染**: Pygame 2.6+ (替代OpenGL)
- **AI框架**: TensorFlow 2.20 + PyTorch 2.4
- **配置管理**: PyYAML 6.0
- **性能监控**: psutil 7.2 + 自研监控器
- **构建工具**: setuptools + pyproject.toml

### 技术选型理由
1. **Pygame替代OpenGL**: 解决Windows ARM64兼容性问题
2. **双AI框架**: 提供更灵活的模型部署选项
3. **轻量级依赖**: 减少系统资源占用
4. **跨平台支持**: 确保在不同环境下的稳定性
