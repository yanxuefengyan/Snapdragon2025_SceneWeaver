# SceneWeaver 菜单系统功能总结

## 📋 概述

SceneWeaver 现已集成完整的菜单栏系统，提供直观的用户界面操作体验。菜单系统包含了文件管理、编辑操作、视图控制、工具和帮助等核心功能。

## 🎨 菜单结构

### 1. 文件菜单 (File)
- **新建** - 创建新项目或场景
- **打开** - 打开现有项目或媒体文件
- **保存** - 保存当前项目
- **另存为** - 将项目保存为新文件
- **退出** - 退出应用程序

### 2. 编辑菜单 (Edit)
- **撤销** - 撤销上一步操作
- **重做** - 重做已撤销的操作
- **剪切** - 剪切选中内容
- **复制** - 复制选中内容
- **粘贴** - 粘贴剪贴板内容

### 3. 视图菜单 (View)
- **全屏** - 切换全屏/窗口模式
- **重置视图** - 重置摄像机视角
- **显示 FPS** - 切换 FPS 计数器显示
- **显示控制台** - 切换控制台显示

### 4. 工具菜单 (Tools)
- **设置** - 打开应用程序设置
- **插件管理** - 管理已安装的插件
- **文件浏览器** - 打开文件选择对话框

### 5. 帮助菜单 (Help)
- **关于** - 显示应用程序信息
- **使用指南** - 打开用户手册
- **检查更新** - 检查软件更新

## ⌨️ 键盘快捷键

## 🔧 技术实现

### 核心组件

1. **MenuSystem** (`src/ui/menu_system.py`)
   - 菜单栏创建和管理
   - 菜单项事件处理
   - 回调函数注册

2. **MenuActions** (`src/ui/menu_system.py`)
   - 所有菜单动作的实现
   - 与引擎和其他系统的集成

3. **GUI系统集成** (`src/ui/gui_system.py`)
   - 菜单栏 UI 元素
   - 下拉菜单支持
   - 事件处理机制

### 使用方法

```python
# 1. 导入菜单系统
from ui.menu_system import MenuSystem, MenuActions

# 2. 初始化菜单
menu_system = MenuSystem(screen_size=(1280, 720))
menu_system.initialize(gui_manager)

# 3. 创建菜单动作并设置回调
menu_actions = MenuActions(engine, gui_system)
menu_system.set_callback('文件', '打开', menu_actions.on_file_open)

# 4. 在主循环中处理事件
for event in pygame.event.get():
    menu_system.process_events(event)
```

## 📁 文件集成

菜单系统已完全集成到主程序中：

- **主程序**: `src/main.py`
  - 自动初始化菜单系统
  - 设置所有菜单回调
  - 连接 GUI 和引擎

- **GUI系统**: `src/ui/gui_system.py`
  - 包含菜单栏组件
  - 提供菜单按钮创建方法
  - 处理菜单事件

- **配置文件**: `config.yaml`
  - UI 启用配置
  - 菜单相关设置

## 💡 使用提示

1. **文件操作**
   - 使用"文件 > 打开"选择图像、视频或配置文件
   - 文件路径会显示在控制面板中
   - 状态栏会显示当前加载的文件状态

2. **视图控制**
   - 按 F11 快速切换全屏模式
   - 使用"视图 > 重置视图"恢复默认视角

3. **帮助功能**
   - 按 F1 查看关于对话框
   - 通过"帮助 > 使用指南"获取使用说明

4. **工具访问**
   - "工具 > 文件浏览器"提供文件选择功能
   - 支持拖拽文件到应用程序窗口

## 📝 开发说明

### 添加新菜单项

1. 在 `MenuSystem._create_menu_bar()` 中添加菜单项定义
2. 在 `MenuActions` 中添加对应的处理方法
3. 在主程序中注册回调函数

### 自定义菜单行为

```python
def my_custom_action():
    print("执行自定义操作")

menu_system.set_callback('工具', '自定义功能', my_custom_action)
```
