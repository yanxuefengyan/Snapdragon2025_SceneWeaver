# SceneWeaver 特效系统详解

## ✨ 特效系统概述

SceneWeaver 配备了强大的实时粒子特效系统，包含 5 种不同的视觉效果：

1. 💥 **爆炸效果** - 强烈的爆炸粒子
2. 💨 **烟雾效果** - 柔和的烟雾扩散
3. 🔥 **火焰效果** - 燃烧的火焰粒子
4. ✨ **魔法效果** - 神秘的魔法光晕
5. ⭐ **火花效果** - 闪烁的火花四溅

所有特效都以 **60 FPS** 实时渲染，提供流畅的视觉体验。

---

## 💥 爆炸效果 (Explosion)

### 触发方式
- 菜单：**特效 (S)** → **爆炸效果**
- 快捷键：无（可通过菜单触发）

### 特效参数
```python
粒子数量：100 个
粒子大小：3-8 像素
速度范围：5-15 像素/帧
持续时间：60 帧（1 秒）
颜色范围：红色/橙色系
```

### 物理特性
- **初始速度**：从中心向四周随机爆发
- **阻力系数**：0.95（逐渐减速）
- **重力影响**：无
- **颜色渐变**：随生命值衰减变暗

### 视觉效果
- 强烈的爆炸冲击感
- 粒子从中心向四周飞散
- 逐渐减速并消失
- 红色到暗红色的颜色过渡

### 使用场景
- 爆炸场景
- 冲击效果
- 强烈视觉冲击

## 🎨 技术实现

### 粒子系统架构

```python
class Particle:
    x, y          # 位置
    vx, vy        # 速度
    radius        # 半径
    color         # 颜色
    life          # 当前生命
    max_life      # 最大生命
```

### 更新循环
```python
1. 更新位置：x += vx, y += vy
2. 应用物理：阻力、重力
3. 减少生命：life -= 1
4. 计算透明度：alpha = life / max_life
5. 绘制粒子
6. 移除死亡粒子
```

### 渲染优化
- ✅ 只绘制存活粒子
- ✅ 批量处理更新
- ✅ 移除死亡粒子释放内存
- ✅ Alpha 混合实现渐变
- ✅ 60 FPS 稳定帧率

## 💡 使用技巧

### 1. 快速触发
- 使用菜单快捷键 **S** 打开特效菜单
- 鼠标点击快速触发火花

### 2. 位置选择
- 爆炸/火花：中心位置效果最佳
- 火焰：底部位置更真实
- 烟雾：任意位置都可
- 魔法：中心汇聚效果好

### 3. 时机把握
- 特效可以叠加触发
- 不同特效可以同时进行
- 注意特效持续时间

### 4. 性能考虑
- 同时触发多个特效会降低 FPS
- 建议依次触发不同特效
- 观察右上角 FPS 监控

## 📝 代码示例

### 触发特效
```python
# 在菜单中触发
elif action == "effect_explosion":
    center_x, center_y = width // 2, height // 2
    for i in range(100):
        angle = random.uniform(0, 2 * math.pi)
        speed = random.uniform(5, 15)
        state["explosion_particles"].append({
            "x": center_x,
            "y": center_y,
            "vx": math.cos(angle) * speed,
            "vy": math.sin(angle) * speed,
            "radius": random.randint(3, 8),
            "color": (255, 100, 0),
            "life": 60,
            "max_life": 60
        })
```
