## 📋 功能概览

所有菜单项已完全实现并可正常使用，涵盖文件管理、编辑操作、视图控制、工具和帮助等五大功能模块。

---

## 🗂️ 文件菜单 (File)

### ✅ 已实现功能

#### 1. 新建 (New)
- **功能**: 创建新项目
- **实现**: 
  - 初始化项目数据结构
  - 设置项目名称为"Untitled"
  - 记录创建时间和修改时间
  - 清空对象和效果列表
- **日志**: `已创建新项目`

#### 2. 打开 (Open)
- **功能**: 打开现有项目文件
- **实现**:
  - 弹出文件选择对话框（tkinter）
  - 支持 .swp 和 .json 格式
  - 加载项目数据到内存
  - 记录当前项目路径
- **日志**: `项目加载成功：{file_path}`

#### 3. 保存 (Save)
- **功能**: 保存当前项目
- **实现**:
  - 如果有路径，直接保存
  - 否则弹出另存为对话框
  - 更新修改时间
  - JSON 格式存储（UTF-8 编码）
- **日志**: `项目保存成功：{file_path}`

#### 4. 另存为 (Save As)
- **功能**: 将项目保存为新文件
- **实现**:
  - 弹出保存对话框
  - 支持自定义文件名和路径
  - 默认扩展名为 .swp
  - 更新当前项目路径
- **日志**: `项目保存成功：{file_path}`

#### 5. 退出 (Exit)
- **功能**: 退出程序
- **实现**:
  - 设置 running = False
  - 触发主循环结束
  - 执行资源清理
- **日志**: `SceneWeaver 正常退出`

## 🎓 使用说明

### 基本操作
1. **启动程序**: `python src/main.py`
2. **访问菜单**: 点击顶部菜单栏
3. **选择项目**: 文件 > 打开
4. **保存工作**: 文件 > 保存
5. **查看帮助**: 帮助 > 使用指南

### 快捷键（计划中）
- Ctrl+N: 新建项目
- Ctrl+O: 打开项目
- Ctrl+S: 保存项目
- Ctrl+Shift+S: 另存为
- F11: 切换全屏
- F1: 显示帮助

---

## 🔧 配置选项

在 `config.yaml` 中可配置：

```yaml
ui:
  enabled: true
  theme: "dark"
  show_performance: true

file_input:
  enabled: true
  max_recent_files: 10
  allow_multiple_selection: true
```
