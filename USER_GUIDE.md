# SceneWeaver 使用指南

## 📖 快速入门

### 1. 启动程序

```bash
python src/main.py
```

程序启动后会显示一个 1280x720 的窗口，包含：
- 顶部菜单栏
- 中央旋转的立方体动画
- "SceneWeaver" 标题

---

## 🎯 菜单功能详解

### 📁 文件菜单 (快捷键：F)

#### 新建项目
1. 点击 **文件** → **新建项目**
2. 创建名为"未命名项目"的新项目
3. 左上角显示项目名称

#### 打开项目
1. 点击 **文件** → **打开项目**
2. 弹出文件选择对话框
3. 选择 `.swp` 或 `.json` 格式的项目文件
4. 点击"打开"加载项目
5. 项目名称显示在左上角

#### 保存项目
1. 点击 **文件** → **保存项目**
2. 弹出保存对话框
3. 选择保存位置和文件名
4. 点击"保存"
5. 项目以 JSON 格式保存（UTF-8 编码，支持中文）

#### 另存为...
1. 点击 **文件** → **另存为...**
2. 选择新的保存位置
3. 输入新文件名
4. 项目保存到新位置，并更新当前项目名称

#### 退出
- 点击 **文件** → **退出**
- 程序显示"再见！"后退出

---

### ✏️ 编辑菜单 (快捷键：E)

#### 撤销
- 点击 **编辑** → **撤销**
- 撤销上一步操作
- 如果无操作可撤销，显示提示

#### 重做
- 点击 **编辑** → **重做**
- 重做已撤销的操作

#### 复制
- 点击 **编辑** → **复制**
- 复制选中内容到剪贴板

#### 粘贴
- 点击 **编辑** → **粘贴**
- 从剪贴板粘贴内容

---

### 👁️ 视图菜单 (快捷键：V)

#### 全屏切换
1. 点击 **视图** → **全屏切换**
2. 程序切换到全屏模式
3. 再次点击恢复窗口模式（1280x720）

#### 显示 FPS
1. 点击 **视图** → **显示 FPS**
2. 右上角显示实时帧率，如："FPS: 60.0"
3. 再次点击隐藏 FPS

#### 重置视图
- 点击 **视图** → **重置视图**
- 恢复默认视图设置

---

### ✨ 特效菜单 (快捷键：S)

#### 爆炸效果 💥
- 点击 **特效** → **爆炸效果**
- 触发爆炸粒子特效
- 底部显示"爆炸效果 💥"

#### 烟雾效果 💨
- 点击 **特效** → **烟雾效果**
- 触发烟雾粒子特效
- 底部显示"烟雾效果 💨"

#### 火焰效果 🔥
- 点击 **特效** → **火焰效果**
- 触发火焰粒子特效
- 底部显示"火焰效果 🔥"

#### 魔法效果 ✨
- 点击 **特效** → **魔法效果**
- 触发魔法粒子特效
- 底部显示"魔法效果 ✨"

#### 火花效果 ⭐
- 点击 **特效** → **火花效果**
- 触发火花粒子特效
- 底部显示"火花效果 ⭐"

---

### ❓ 帮助菜单 (快捷键：H)

#### 使用说明
1. 点击 **帮助** → **使用说明**
2. 显示详细的帮助对话框（750x550）
3. 包含所有菜单功能的详细说明
4. 关闭方式：
   - 点击右上角 × 按钮
   - 点击对话框外部
   - 按 ESC 键

#### 关于
1. 点击 **帮助** → **关于**
2. 显示软件信息对话框（600x450）
3. 包含：
   - 软件名称和版本
   - 构建日期
   - 核心功能列表
   - 版权信息
4. 关闭方式：
   - 点击右上角 × 按钮
   - 点击对话框外部
   - 按 ESC 键

## 💡 使用技巧

### 1. 快速访问菜单
- 使用键盘快捷键 **F/E/V/S/H** 快速打开对应菜单
- 比鼠标点击更快捷

### 2. 项目管理
- 项目文件使用 JSON 格式，易于编辑和分享
- 支持 UTF-8 编码，完美支持中文
- 可以手动编辑 JSON 文件来自定义项目

### 3. 性能监控
- 开启 FPS 显示实时监控性能
- 正常应该保持在 60 FPS
- 如果 FPS 过低，关闭一些特效

### 4. 全屏体验
- 全屏模式提供更沉浸的视觉体验
- 按 ESC 退出全屏

### 5. 特效演示
- 依次点击不同特效查看效果
- 可以组合使用多个特效
- 特效会显示在左上角状态栏

## 📂 项目文件格式

### 示例项目文件 (.swp)
```json
{
  "name": "我的场景项目",
  "created": "2026-03-07",
  "settings": {
    "width": 1280,
    "height": 720,
    "effects_enabled": true,
    "show_fps": false
  },
  "scenes": [],
  "metadata": {
    "author": "用户名",
    "version": "1.0"
  }
}
```

### 手动创建项目
1. 用文本编辑器创建 `.json` 文件
2. 按照上述格式编写内容
3. 保存为 `.swp` 或 `.json` 扩展名
4. 在程序中点击"打开项目"加载

## 🔧 常见问题

### Q: 窗口打不开怎么办？
A: 检查以下几点：
1. 确保已安装 pygame: `pip install pygame`
2. 确保 Python 版本 >= 3.8
3. 查看控制台是否有错误信息
4. 尝试以管理员身份运行

### Q: 菜单显示方框而不是中文？
A: 字体问题，解决方案：
1. 确保 Windows 系统安装了中文字体
2. 程序会自动检测并使用黑体/宋体/微软雅黑
3. 如果仍然显示方框，重启程序

### Q: 如何保存项目？
A: 
1. 点击 **文件** → **保存项目**
2. 选择保存位置
3. 输入文件名（建议使用 .swp 扩展名）
4. 点击保存

### Q: 特效不显示怎么办？
A: 
1. 检查是否开启了特效（查看左上角状态）
2. 尝试重新触发特效
3. 查看控制台是否有错误信息

### Q: 如何退出全屏模式？
A: 
- 方法 1: 点击 **视图** → **全屏切换**
- 方法 2: 按 **ESC** 键

### Q: 对话框关不掉？
A: 
- 点击对话框外部的任意区域
- 或按 **ESC** 键
- 或点击右上角的 × 按钮

## 🚀 进阶使用

### 自定义项目数据
1. 打开或创建项目
2. 手动编辑 JSON 文件添加自定义字段
3. 保存后在程序中加载

### 扩展菜单功能
修改 `src/main.py` 中的 `menu_items` 列表：
```python
menu_items = [
    {"text": "文件 (F)", "key": pygame.K_f, "submenu": [
        {"text": "你的功能", "action": "your_action"},
        # ... 更多功能
    ]},
    # ... 更多菜单
]
```

然后在 `handle_menu_action` 函数中添加处理逻辑。
